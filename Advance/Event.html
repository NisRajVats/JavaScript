3. Info

  • Event Bubbling: Event propagates from the innermost element up to the outer elements (default).
  • Event Capturing: Event propagates from the outermost element down to the target element (use by passing true as the third argument to addEventListener).
  • stopPropagation(): Stops the event from further propagation.
  • preventDefault(): Prevents the default action (e.g., form submission).
